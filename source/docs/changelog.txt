VERSION 0.1.0 - Beta Release
* Beta version of StudyAid. Releases will be labeled with the following format - #.#.#
  The first digit will represent a major build update (i.e. framework update, layout, multiple features etc.).
  The second digit will represent moderate updates (i.e. features, bug fixes, optimization updates etc.).
  The third digit will represent light updates (i.e. bug fixes and/or small optimization updates).
